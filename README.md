## 1 需求的发散与收敛

### 发散阶段

3 个关键点：层层分解，上层概况下次，同层独立穷尽  
5 个维度：功能，性能，场景，API，设计约束

### 收敛阶段

3 个关键：重要性，复杂度，影响面

步骤

1. 筛选整合
2. 优先级排序
3. 划定边界
4. 确定可行性
